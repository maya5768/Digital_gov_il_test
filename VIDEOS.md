## 🔧 כיצד להריץ הבדיקות בעצמך

### התקנה:
```bash
npm install cypress --save-dev
```

### הרצת בדיקה ספציפית:
```bash
# בדיקה 1: רכיבי חיפוש
npx cypress run --spec "cypress/e2e/1-search-component-tests.cy.js"

# בדיקה 2: יירוט בקשות
npx cypress run --spec "cypress/e2e/2-intercept-requests.cy.js"

# בדיקה 3: זיהוי תורים
npx cypress run --spec "cypress/e2e/3-services-appointment-check.cy.js"

# בדיקה 3א: רכיבי UI
npx cypress run --spec "cypress/e2e/3-components-appointment-ui-test.cy.js"

# בדיקה 4: שינוי API
npx cypress run --spec "cypress/e2e/4-mainbanner-response-change.cy.js"
```

### הרצה עם ממשק גרפי:
```bash
npx cypress open
```
